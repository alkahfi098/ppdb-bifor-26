Rekayasa Perangkat Lunak (RPL)

Jurusan Rekayasa Perangkat Lunak (RPL) adalah jurusan yang mempelajari dan mendalami cara-cara mengembangkan perangkat lunak atau software. Di jurusan ini, siswa akan belajar membuat aplikasi, game, website, hingga sistem informasi yang bisa digunakan di dunia nyata.

Apa yang Dipelajari di RPL?

  • Pemrograman dasar (HTML, CSS, JavaScript, Python, Java, C++)
  • Pembuatan dan pengembangan aplikasi berbasis web & mobile
  • Database dan manajemen data (MySQL, Firebase, dsb.)
  • Desain UI/UX dan alur sistem
  • Pengujian software (software testing)
  • Pengembangan game sederhana
  • Penggunaan tools pengembangan seperti Visual Studio Code, Android Studio, GitHub

Apa yang Dikerjakan Siswa RPL?

  • Membuat website sekolah, toko online, dan blog pribadi
  • Merancang dan membuat aplikasi Android sederhana
  • Membuat game edukasi menggunakan Unity atau Construct
  • Membuat sistem informasi akademik, kasir, inventaris, dll
  • Berpartisipasi dalam lomba IT seperti LKS, Hackathon, atau Festival Komputer
  • Bekerja dalam tim seperti layaknya di dunia kerja (scrum, version control)

Peluang dan Prospek Kerja

  • Web Developer
  • Mobile App Developer
  • Game Developer
  • Software Engineer
  • UI/UX Designer
  • IT Support / Programmer di perusahaan atau startup
  • Freelancer IT / Digital Entrepreneur

RPL sangat cocok bagi kamu yang suka logika, berpikir kreatif, dan tertarik dengan dunia teknologi.
